# GPT sin Alzheimer 🧠💾

> Tu asistente GPT local, con memoria extendida y cero olvidos.

**GPT sin Alzheimer** es una aplicación de escritorio que corre localmente y permite extender la memoria de ChatGPT de forma segura, sin conexión a la nube y sin necesidad de conocimientos técnicos.

## 🧩 Funciones

- Multiusuario por carpetas
- Configuración y notas modificables desde el propio GPT
- Ejecutable `.exe` con icono y bandeja de sistema
- Almacenamiento flexible con soporte Markdown y emojis
- Ideal para mundos de campaña, escritores, y proyectos personales

## 🛠 Cómo usar

1. Ejecuta el `.exe`
2. Selecciona usuario o crea uno nuevo
3. Chatea con tu GPT personalizado y empieza a guardar, editar y recordar
